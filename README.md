You can also use private nuget feed from [AppVeyor CI builds](https://ci.appveyor.com/nuget/sharputils-fsrqkyoeqb08).

## Building and testing
Use `dotnet` for building, creating nuget and running tests:
```
dotnet build
dotnet pack
dotnet test
```
